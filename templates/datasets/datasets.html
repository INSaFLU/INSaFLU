*Please make sure you include a “date” (format: YYYY-MM-DD) for all samples. If no date is provided, INSaFLU will automatically insert the date of the analysis as the “collection date”, which might (considerably) bias (or even break) the time-scale trees generated for influenza, SARS-CoV-2 and Monkeypox.

If you provide “region, country, division and/or location”, these values will be screened against a vast database of “latitude and longitude” coordinates to geographically place the sequences in the Nextstrain map.
