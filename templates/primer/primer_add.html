Guide to generate new primer sets

If you want to upload additional primer sets to the user-restricted primer database.

  • You need to upload a multi-FASTA file with the set of primer sequences. Example:

    >Forward1
    AGTCGTACGT
    >Reverse1
    CGTGACAGTGA
    >Forward2
    TTGCAGTTTT
    >Reverse2
    GGTGAACGTCA

  • You also need to upload a tabular file with the primer pairs. Example:

    Forward1 Reverse1
    Forward2 Reverse2
